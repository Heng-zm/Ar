# **App Name**: ARchitect

## Core Features:

- Camera Activation: Enable and display the device's camera feed in the web application.
- 3D Model Import: Allow users to import 3D models from various file formats (.glb, .gltf).
- AR Placement and Manipulation: Enable users to place, scale, rotate, and position 3D models within the live AR view.
- Scene Capture: Provide functionality to capture a screenshot of the composite AR scene.

## Style Guidelines:

- Primary color: Electric blue (#7DF9FF) to highlight interactive elements.
- Background color: Dark grey (#222222) to create a modern, immersive environment.
- Accent color: Lighter grey (#AAAAAA) for text and secondary UI elements to ensure readability against the dark background.
- Font: 'Inter', a sans-serif font, for both headlines and body text due to its clean, modern appearance suitable for UI elements.
- Use minimalist, vector-based icons with electric blue accents to maintain a consistent modern aesthetic.
- Implement a clean, intuitive layout that prioritizes the camera view, with easily accessible controls for model manipulation and import.
- Use subtle animations for user interactions, such as object placement and scaling, to provide feedback without distracting from the AR experience.
